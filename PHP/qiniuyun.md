# [七牛云SDK](http://developer.qiniu.com/code/v7/sdk/php.html)
##thinkphp3.2.3集成七牛云SDK
###1. 安装七牛云SDK，然后引入（请参考[thinkphp3.2.3引入第三方包](./tpkuang_jia_yin_ru_di_san_fang_bao.md)）
###2. 配置七牛云参数
```php
//七牛云配置
    'accessKey' => 'XXX',//秘钥
    'secrectKey' => 'XXX',//秘钥
    'domain' => 'o7atl50ri.bkt.clouddn.com',//域名，这里是七牛云提供的测试域名
    'bucket' => 'XX',//'<空间名称>',
```
