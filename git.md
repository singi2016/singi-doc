# git

###克隆一个新项目
```
git clone https://github.com/singi2016cn/webpack-start.git
```

### 查看所有分支
```
git branch -a
```

### 从远端新建本地分支并切换过去
```
git checkout -b branch origin/branch
```

### 添加文件或目录到暂存区
```
git add <file>
```
### 取消已添加到暂存区的文件或目录
```
git checkout -- <file>
```

### 提交添加的文件到本地仓库
```
git commit -m <message>
```

### 取消commit到本地仓库
```
git rm <file>
```

### 提交本地仓库的文件到远端仓库
```
git push
```
### 更新远端仓库到本地
```
git pull
```
